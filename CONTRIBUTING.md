# Contributing

Thank you for your interest in improving the K1 Wizard-of-Oz Dashboard for
the broader educator community.

---

## Who should contribute

- Faculty and staff at colleges using this project in their AI curriculum
- Developers who work with the Booster K1 EDU or similar humanoid robots
- Educators building on the AI PREP4WORK Initiative model

---

## How to contribute

### Reporting a bug

Open an issue at https://github.com/TechPlayzone/k1-wizard-of-oz/issues

Please include:
- Your OS, Python version, and K1 firmware version
- Steps to reproduce the problem
- What you expected to happen vs. what actually happened
- Relevant terminal output or error messages

### Suggesting an improvement

Open an issue with the label `enhancement`. Describe:
- What you are trying to accomplish
- Why the current project does not support it
- Any ideas you have for implementation

### Submitting a pull request

1. Fork the repository.
2. Create a branch: `git checkout -b fix/your-description`
3. Make your changes.
4. Test against a live K1 or with Ollama locally.
5. Submit a pull request with a clear description of the change.

---

## Code style

- Python: follow PEP 8. Use `snake_case` for variables and functions.
- HTML/JS: 2-space indentation. Keep the dashboard as a single `index.html`
  file to preserve ease of deployment at partner colleges.
- Comments: write them as if explaining to an educator, not just a developer.
